# Vue-tube

> Vue.js를 이용한 영화 정보 제공 사이트 제작 프로젝트



<br>

<br>

## 1. 개발 환경

- Vue 개발 환경
  - Node 12.16.3
  - Vue 2.6.11
  - Vue CLI 4.4.0

<br>
- 서비스 배포 환경
  - Netlify (https://vue-tuube.netlify.app/)



<br>

<br>

## 2. 기능

- **영화 정보 제공 기능**
  - 영화 가져오기 버튼을 누르면 데이터를 비동기적으로 받아옵니다
- **영화 예고편 검색 기능**
  - 영화 제목을 검색하면 Youtube API를 통해 데이터를 비동기적으로 받아옵니다
- **Scroll to top 기능**
  - 우측 하단의 top 버튼을 클릭하면 바로 화면의 제일 위로 이동할 수 있습니다
